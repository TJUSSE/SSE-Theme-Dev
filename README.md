# Tongji SSE Front-end Dev Repo

## Requires

Please install nodejs (version >= 0.12) and bower, gulp are required.

Then
```bash
$ git clone https://github.com/TJUSSE/SSE-Theme-Dev.git
$ cd SSE-Theme-Dev
$ npm install
$ bower install
```

## How to develop

In the repo directory,
```bash
$ gulp watch
```
